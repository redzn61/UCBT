Project Vision

Why this website was created

This platform was created to present the university in a modern way and to give students a useful digital space where they can access courses, marks, absences, timetable, departments, news and university information.

🎯 Main Objective

Build a complete university website with professional UI/UX, responsive design, student portal, admin dashboard and PHP backend.

🎨

Design Level

Modern glassmorphism, 3D animated cards, glowing borders, futuristic gradients, hover movement and responsive layouts.

⚙️

Technologies

  • HTML5 pages
  • CSS3 animations and 3D effects
  • PHP authentication
  • JSON student database
  • Admin dashboard
🚀

Features

  • Student login system
  • Dynamic notes and absences
  • Timetable display
  • Top 5 students ranking
  • Contact form backend
